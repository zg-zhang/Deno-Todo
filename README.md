# Deno-Todo

🦕 Deno 正式版终于发布了，支持 TypeScript 和 ES6 真的是太好啦。

## run

```
cd ./Deno-Todo
$ deno run -A ./index.ts 

// 重新编译
$ deno run -A --reload ./index.ts
```
